# Appium Android Driver

Appium Android Driver is the collection of common primitives used by other Android drivers, like UIA2/Espresso. This driver cannot work as a separate entity.

## Development

### Install and Compile

```
npm i
npm run build
npm run lint
```

### Unit Test

```
npm test
```
